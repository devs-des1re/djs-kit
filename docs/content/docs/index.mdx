Welcome to **djs-kit**! The fastest and most robust way to start building modern Discord bots with [discord.js v14](https://discord.js.org/).

## Why djs-kit?

Building a Discord bot from scratch requires a lot of boilerplate: command handlers, event loaders, component routers, state management, and strict TypeScript configurations.

**djs-kit solves all of this by providing:**
- **Instant Scaffolding:** Create a fully-functioning bot in seconds with interactive prompts.
- **Type-Safe Builders:** Chainable, deeply typed builder APIs for commands and components.
- **Signed Component State:** Pass variables between commands and components with expiring, scoped custom IDs.
- **Event Loader:** Keep Discord events in their own folder, loaded automatically.
- **Database Presets:** Choose memory, file, SQLite, Postgres, MySQL, MongoDB, or Redis.
- **Organized Architecture:** A clean, modular folder structure that scales seamlessly.
- **TypeScript & JavaScript Support:** Choose between strict TS or flexible JS output.

## Quick Start

You can generate a new project using `npx` directly from your terminal:

```bash
npx @devs-des1re/djs-kit create my-discord-bot
```

The CLI will launch an interactive flow to configure your bot:

1. **Output Language:** Choose TypeScript or JavaScript.
2. **Bot Token:** Paste your Discord bot token (it validates it in real-time!).
3. **Client & Guild IDs:** Used to register slash commands immediately on startup.
4. **Database Preset:** Choose how cooldowns and storage should be scaffolded.
5. **Project Preset:** Start bare or generate utility, moderation, tickets, or community examples.

## Directory Structure

Once scaffolded, your project will have a beautifully structured setup:

```text
my-discord-bot/
├── src/
│   ├── builders/        # Type-safe Builder APIs
│   ├── autocomplete/    # Autocomplete handlers
│   ├── commands/        # Your Slash & Prefix Commands
│   ├── components/      # Your Buttons, Modals, and Select Menus
│   ├── context/         # User and message context menus
│   ├── events/          # Discord client events
│   ├── handlers/        # Automated Loaders and Routers
│   ├── lib/             # Utilities, logging, responses, custom IDs
│   ├── config.ts        # Bot behavior config
│   └── index.ts         # Bot Entry Point
├── package.json
└── tsconfig.json        # (If using TypeScript)
```

## Running Your Bot

To start your bot in development mode, simply run:

```bash
cd my-discord-bot
npm run dev
```

`djs-kit` comes pre-configured with a **live-reloading development server**. Whenever you make changes to a file or add a new command, your bot will automatically restart and load the new changes instantly!
